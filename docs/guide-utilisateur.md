## 1. Introduction

Le **portail Carte Promote** est la plateforme numérique d'Afriland First Bank pour la gestion complète du programme de carte prépayée **Carte Promote**. Il remplace les formulaires papier et les fichiers Excel par un parcours entièrement numérique, accessible depuis tout smartphone ou ordinateur connecté à Internet.

### Ce que permet le portail

| Pour qui | Ce qu'il peut faire |
|---|---|
| **Client** | Souscrire à la carte, suivre sa demande, recharger sa carte |
| **Agent commercial** | Souscrire pour un client, gérer ses ventes, suivre ses performances |
| **Caissier** | Valider les paiements espèces et virements SARA, créditer les recharges |
| **Point d'impression** | Vérifier les dossiers KYC, imprimer et remettre les cartes |
| **Collecteur terrain** | Saisir les ventes de produits bancaires |
| **Superviseur** | Consulter les statistiques de collecte |
| **Administrateur** | Gérer les utilisateurs, la configuration, les statistiques globales |

### Navigateurs compatibles

Le portail fonctionne sur tous les navigateurs modernes : **Chrome**, **Firefox**, **Edge**, **Safari**. Il est optimisé pour les **smartphones Android et iOS**.

## 2. Accès au portail

### URL d'accès

L'URL du portail vous est fournie par votre responsable Afriland. Elle ressemble à :

```
https://promote.afrilandfirstbank.com
```

### Accès clients (sans compte)

Les clients accèdent directement au formulaire de souscription via :
- Un **QR code** fourni par un agent (le scan ouvre automatiquement le formulaire)
- L'URL de **souscription directe** communiquée par l'agence
- La page **Recharge** pour recharger une carte existante

### Accès staff (avec compte)

Le personnel d'Afriland se connecte via la page de connexion. Les identifiants sont fournis par l'administrateur. Voir [section 11](#11-connexion-et-gestion-du-compte-staff).

## 3. Guide client — Souscrire à la Carte Promote

### Avant de commencer

Préparez les éléments suivants :
- Votre **carte nationale d'identité** (CNI) valide — recto et verso
- Votre **numéro de téléphone Mobile Money** (Orange Money ou MTN) si vous souhaitez payer par Mobile Money
- Un endroit bien éclairé pour la photo (selfie)

### Étape 1 — Identité

Remplissez vos informations personnelles :

| Champ | Description |
|---|---|
| Prénom / Nom | Tels qu'ils apparaissent sur votre CNI |
| Sexe | Masculin / Féminin |
| Type de pièce | Carte Nationale d'Identité, Passeport, etc. |
| N° de pièce | Numéro figurant sur votre document |
| NIU | Numéro d'Identification Unique (si disponible) |
| Date d'expiration | Date de fin de validité de votre pièce (format JJ/MM/AAAA) |
| Téléphone | Votre numéro camerounais (ex. 699 000 000) |
| E-mail | Votre adresse e-mail (optionnel) |
| Quartier / Ville | Votre adresse de résidence |

Cliquez sur **Suivant** quand tous les champs obligatoires (marqués \*) sont remplis.

### Étape 2 — Documents d'identité

Photographiez votre carte d'identité :

1. Cliquez sur **Photographier le recto** → pointez votre caméra sur la face avant de votre CNI → cliquez sur le bouton de capture.
2. Vérifiez que la photo est nette et lisible.
3. Répétez pour le **verso** (face arrière).

> **Conseil :** Posez la CNI à plat sur une surface sombre, éclairage frontal, évitez les reflets.

Cliquez sur **Suivant** une fois les deux photos capturées.

### Étape 3 — Votre photo (Selfie)

Cette étape est **obligatoire** : votre photo servira à vérifier votre identité lors de la remise de la carte.

1. Cliquez sur **Prendre ma photo**.
2. Regardez droit vers la caméra, visage bien éclairé, sans lunettes de soleil.
3. Cliquez sur le bouton de capture.
4. Si la photo ne vous convient pas, cliquez sur **Reprendre**.

Cliquez sur **Suivant** une fois votre selfie validé.

### Étape 4 — Paiement et livraison

#### Type de carte

- **Carte Prépayée** : carte rechargeable (usage courant)
- **Carte Bancaire** : carte liée à un compte bancaire

#### Mode de livraison

| Option | Description |
|---|---|
| Point Promote | Retrait dans un point d'impression Promote (délai : 2-5 jours ouvrés) |
| Agence Afriland | Retrait dans l'agence de votre choix |
| Livraison à domicile | Livraison à votre adresse (frais de transport applicables) |

#### Méthode de paiement

| Méthode | Comment faire |
|---|---|
| **Orange Money** | Saisissez votre numéro Orange. Vous recevrez une invite USSD sur votre téléphone à confirmer. |
| **MTN Mobile Money** | Saisissez votre numéro MTN. Idem. |
| **Espèces** | Payez directement au guichet d'une agence Afriland. Un caissier validera votre paiement. |
| **Virement SARA** | Effectuez un virement SARA et uploadez votre reçu (photo ou PDF). |

#### Référent commercial (optionnel)

Si un agent commercial vous a recommandé, saisissez son numéro de téléphone dans le champ **Référent**.

### Étape 5 — Récapitulatif et confirmation

Vérifiez toutes vos informations. Si tout est correct, cliquez sur **Confirmer ma souscription**.

### Après la souscription

Vous obtenez votre **référence de souscription** (ex. `PRM-1042`) et un **QR code** de votre dossier.

- **Paiement Mobile Money** : confirmez l'invite USSD sur votre téléphone. La page se met à jour automatiquement.
- **Paiement espèces** : rendez-vous dans une agence Afriland avec votre référence.
- **Virement SARA** : un agent validera votre reçu dans les 24h ouvrées.

> **Important :** Conservez votre référence `PRM-XXXX`. Elle est nécessaire pour suivre votre dossier et retirer votre carte.

## 4. Guide client — Recharger sa carte

### Accès

Depuis la page d'accueil du portail, cliquez sur **Recharger ma carte**.

### Formulaire de recharge

| Champ | Description |
|---|---|
| Prénom / Nom | Votre identité (détenteur de la carte) |
| Téléphone | Votre numéro de téléphone |
| N° de carte (4 premiers chiffres) | Premiers chiffres du PAN de votre carte |
| N° de carte (4 derniers chiffres) | Derniers chiffres du PAN de votre carte |
| Montant | Montant à recharger (entre les bornes affichées) |
| Méthode de paiement | Orange Money, MTN, Espèces ou SARA |

Après soumission, suivez les mêmes étapes de paiement que pour une souscription.

> **Note :** La recharge apparaît sur votre carte après validation par un caissier (généralement dans la journée).

## 5. Guide agent commercial

### Accès et connexion

Connectez-vous avec votre e-mail et mot de passe Afriland. Vous arrivez sur votre **tableau de bord agent**.

### Tableau de bord

Votre tableau de bord affiche :
- **Vos souscriptions** du jour / de la semaine / du mois (nombre, montant total)
- **Vos souscriptions récentes** avec leur statut
- **Votre classement** dans le réseau commercial

### Créer une souscription pour un client

1. Cliquez sur **Nouvelle souscription** dans la barre de navigation.
2. Remplissez le formulaire en 5 étapes à la place du client (voir [section 3](#3-guide-client--souscrire-à-la-carte-promote) pour le détail des champs).
3. À l'étape **Paiement**, si un collègue vous a référé le client, saisissez son numéro dans le champ **Référent**.
4. Confirmez. La souscription est automatiquement rattachée à votre compte.

### Consulter vos souscriptions

1. Dans le menu, cliquez sur **Mes souscriptions**.
2. Vous voyez la liste de toutes vos souscriptions avec les indicateurs de statut :
   - 🕐 **En attente** : paiement en cours
   - ✅ **Payée** : paiement confirmé, en attente d'impression
   - 🖨️ **Imprimée** : carte remise au client
   - ❌ **Échec** : paiement échoué

3. Utilisez les filtres (date, statut) pour affiner la liste.

### Récupérer une souscription QR (réclamation)

Si un client a souscrit lui-même via votre QR code :
1. Dans **Mes souscriptions**, cliquez sur **Récupérer une souscription**.
2. Scannez le QR code de la confirmation du client ou saisissez sa référence.
3. La souscription est créditée dans vos statistiques.

### Valider un reçu SARA

Si un client a effectué un virement SARA :
1. Recherchez sa souscription (statut `Sara en attente`).
2. Consultez le reçu uploadé et les données extraites par OCR.
3. Vérifiez que le montant et la référence correspondent.
4. Cliquez sur **Valider** ou **Rejeter**.

### Partager votre QR code

Votre QR code personnel est accessible depuis votre profil. Vous pouvez le partager (WhatsApp, impression) pour que des clients souscrivent directement et soient rattachés à vous.

## 6. Guide caissier

### Accès et connexion

Connectez-vous avec vos identifiants. Vous arrivez sur le **tableau de bord caissier**.

### Valider un paiement espèces

1. Dans le menu, cliquez sur **Paiements en attente** → filtre **Espèces**.
2. La liste affiche toutes les souscriptions et recharges en attente de paiement espèces.
3. Sélectionnez un dossier.
4. Vérifiez l'identité du client et le montant dû.
5. Encaissez les espèces physiquement.
6. Dans le portail, saisissez la **référence du reçu papier** que vous remettez au client.
7. Cliquez sur **Valider le paiement**. Le dossier passe en statut `Payé`.

Pour **rejeter** un paiement (client absent, montant incorrect) : cliquez sur **Rejeter** et saisissez le motif.

### Valider un reçu SARA

1. Dans **Paiements en attente** → filtre **SARA**.
2. Ouvrez le dossier. Consultez l'image du reçu et les données extraites automatiquement.
3. Vérifiez : référence SARA, montant, numéro de téléphone du payeur.
4. Cliquez sur **Valider** ou **Rejeter**.

### Créditer une recharge (Fulfillment)

1. Dans le menu, cliquez sur **Recharges à créditer**.
2. La liste affiche les recharges payées non encore créditées.
3. Sélectionnez une recharge.
4. Effectuez le crédit de la carte dans votre système de gestion des cartes (opération externe au portail).
5. Une fois le crédit effectué, dans le portail, cliquez sur **Confirmer le crédit**.
6. Le dossier est marqué `Créditée` avec votre nom et l'heure.

## 7. Guide point d'impression

### Accès et connexion

Connectez-vous avec vos identifiants. Vous arrivez sur le **tableau de bord point d'impression**.

### Liste des dossiers à imprimer

La page d'accueil affiche les souscriptions payées en attente d'impression, triées par date.

### Vérifier un dossier KYC

1. Cliquez sur un dossier ou recherchez-le par référence, nom ou téléphone.
2. Le dossier affiche :
   - Informations personnelles du client
   - **Selfie** (agrandissable)
   - **CNI Recto** et **CNI Verso** (agrandissables)
   - Statut du selfie (validé / non validé)

3. Vérifiez visuellement que le selfie correspond à la personne qui se présente devant vous.

### Valider ou rejeter le selfie

- Cliquez sur **Valider le selfie** si la photo correspond à la personne présente.
- Cliquez sur **Rejeter le selfie** si elle ne correspond pas (fraude potentielle). Consignez le motif.

### Remplacer une photo défectueuse

Si une photo est floue ou illisible :
1. Dans le dossier, cliquez sur **Remplacer la photo** à côté du document concerné.
2. Capturez ou uploadez la nouvelle photo.
3. Confirmez le remplacement.

### Imprimer et remettre la carte

1. Une fois le KYC vérifié, imprimez physiquement la carte sur votre machine d'impression.
2. Saisissez dans le portail le **numéro de la carte** remise :
   - **4 premiers chiffres** dans le premier champ
   - **4 derniers chiffres** dans le second champ
3. Cliquez sur **Marquer comme imprimée**.
4. Remettez la carte au client.

> Le portail enregistre automatiquement votre nom, votre identifiant et l'heure exacte de la remise. Cette information est immuable.

## 8. Guide collecteur terrain

### Accès et connexion

Connectez-vous avec vos identifiants. Vous arrivez sur votre **interface de collecte**.

### Saisir une nouvelle collecte

1. Cliquez sur **Nouvelle collecte**.
2. Sélectionnez le **produit vendu** :

| Produit | Informations complémentaires à saisir |
|---|---|
| Compte ouvert | Numéro de compte bancaire |
| Carte bancaire | Numéro de carte (masqué, 4+4 chiffres), type de carte |
| SARA Money | Informations du client |
| e-First | Informations du client |

3. Saisissez le **nom** et le **téléphone** du client.
4. Cliquez sur **Enregistrer**. La collecte est enregistrée sous la référence `COL-XXXXXX`.

### Consulter vos collectes

Depuis la page principale, vous visualisez l'historique de toutes vos collectes avec les filtres date et produit.

## 9. Guide superviseur collecte

### Accès et connexion

Connectez-vous avec vos identifiants. Vous accédez au **tableau de bord superviseur**.

### Consulter les statistiques

Le tableau de bord affiche :
- Nombre de collectes par type de produit
- Nombre de collectes par collecteur

### Filtrer les données

Utilisez les filtres pour affiner la vue :
- **Période** : dates de début et de fin

### Exporter les données

Cliquez sur **Exporter Excel** pour télécharger les données filtrées dans un fichier `.xlsx`. Ce fichier peut être ouvert avec Microsoft Excel ou LibreOffice Calc.

## 10. Guide administrateur

### Accès et connexion

Connectez-vous avec vos identifiants administrateur. Vous arrivez sur le **tableau de bord administrateur** — vue globale de toute l'activité.

### Tableau de bord global

- **Souscriptions du jour / mois / total** : par statut (en attente, payées, imprimées, échecs)
- **Volumes encaissés** : par méthode de paiement
- **Top agents** : classement des agents par nombre de ventes
- **Statistiques par agence**

### Gestion des utilisateurs

Menu **Utilisateurs** → liste de tous les comptes staff.

#### Créer un compte

1. Cliquez sur **Nouveau utilisateur**.
2. Remplissez : nom, e-mail, téléphone, agence, rôle(s), mot de passe initial.
3. Cliquez sur **Créer**. L'utilisateur devra changer son mot de passe à la première connexion.

#### Rôles disponibles

| Rôle | Accès |
|---|---|
| `ADMIN` | Tout le portail |
| `AGENT` | Souscriptions, tableau de bord agent |
| `PRINT_AGENT` | Point d'impression |
| `CASHIER` | Caisse, recharges |
| `COLLECTEUR` | Collecte terrain |
| `SUPERVISEUR` | Statistiques collecte (lecture seule) |

Un utilisateur peut avoir **plusieurs rôles** simultanément.

#### Modifier / Désactiver un compte

1. Cliquez sur l'utilisateur dans la liste.
2. Modifiez les informations souhaitées.
3. Pour désactiver : décochez **Compte actif**. L'utilisateur ne pourra plus se connecter.

#### Réinitialiser les credentials

1. Sélectionnez l'utilisateur.
2. Cliquez sur **Réinitialiser le mot de passe** ou **Réinitialiser le PIN**.
3. Définissez le nouveau mot de passe temporaire et communiquez-le à l'utilisateur.

#### Import en masse

1. Préparez un fichier CSV/Excel avec les colonnes : nom, e-mail, téléphone, agence, rôle.
2. Cliquez sur **Importer** et téléversez le fichier.
3. Le système importe les nouveaux comptes et ignore les e-mails déjà enregistrés.

### Configuration tarifaire

Menu **Configuration** :

| Paramètre | Description |
|---|---|
| Prix de la carte | Montant en XAF facturé au client pour la carte |
| Frais de service | Frais additionnels |
| Frais de transport | Frais de livraison à domicile |
| Montant min recharge | Recharge minimale autorisée |
| Montant max recharge | Recharge maximale autorisée |

> **Attention :** toute modification s'applique immédiatement à toutes les nouvelles souscriptions.

### Gestion des agences

Menu **Agences** : ajoutez, modifiez ou désactivez des agences. Vous pouvez importer la liste des agences depuis un fichier Excel.

### Toutes les souscriptions

Menu **Souscriptions** : liste complète filtrée et paginée de toutes les souscriptions (toutes agences, tous agents confondus).

Filtres disponibles : statut, méthode de paiement, agence, agent, date, canal (agent/self).

Chaque souscription est consultable en détail : informations client, documents KYC, historique de paiement.

### Réconciliation des paiements

Si des paiements Mobile Money restent bloqués en statut `En attente` :
1. Menu **Paiements** → **Réconciliation manuelle**.
2. Saisissez la fenêtre temporelle (ex. dernières 24h).
3. Cliquez sur **Lancer la réconciliation**.
4. Le système interroge TrustPayWay et met à jour les statuts.
5. Un résumé s'affiche : `Vérifiés / Mis à jour / Inchangés`.

### Journal d'audit

Menu **Audit** :

- **Actions** : toutes les créations, modifications, validations effectuées sur le portail, avec l'acteur, la date et l'adresse IP.
- **Connexions** : toutes les tentatives de connexion (succès et échecs).

Utilisez les filtres pour cibler une investigation.

### Carte de géolocalisation

Menu **Carte** : visualisez en temps quasi réel la position des agents terrain connectés.

Chaque point sur la carte indique : nom de l'agent, agence, heure de la dernière position transmise.

### Export des données

Depuis la liste des souscriptions ou des recharges, cliquez sur **Exporter Excel** pour télécharger les données filtrées.

### Notifications

Vous pouvez envoyer des notifications in-app à des utilisateurs spécifiques ou à un groupe. Menu **Notifications** → **Nouvelle notification**.

## 11. Connexion et gestion du compte (staff)

### Se connecter

1. Accédez à la page de connexion du portail.
2. Choisissez votre méthode de connexion :
   - **E-mail + mot de passe** : saisissez votre adresse e-mail professionnelle Afriland et votre mot de passe.
   - **Téléphone + PIN** : saisissez votre numéro de téléphone et votre code PIN à 4 chiffres (usage mobile).
3. Cliquez sur **Se connecter**.

### Premier login — Changement de mot de passe obligatoire

Lors de votre première connexion, vous serez automatiquement redirigé vers l'écran de changement de mot de passe. Définissez un mot de passe fort (minimum 8 caractères, combinant lettres et chiffres) puis cliquez sur **Valider**.

### Mot de passe oublié

1. Sur la page de connexion, cliquez sur **Mot de passe oublié**.
2. Saisissez votre adresse e-mail professionnelle.
3. Vous recevrez un e-mail avec un lien de réinitialisation (valable 1h).
4. Cliquez sur le lien, définissez un nouveau mot de passe.

> Si vous ne recevez pas l'e-mail, vérifiez vos spams ou contactez votre administrateur.

### Se déconnecter

Cliquez sur votre nom / avatar en haut à droite → **Se déconnecter**. Votre session expire automatiquement après 24h d'inactivité.

## 12. Questions fréquentes

**Q : Mon paiement Mobile Money est resté « En attente » depuis plus de 30 minutes. Que faire ?**

Contactez votre agence Afriland en fournissant votre référence (`PRM-XXXX`). Un administrateur peut déclencher une réconciliation manuelle pour vérifier l'état réel du paiement auprès de l'opérateur.

---

**Q : J'ai souscrit mais je n'ai pas reçu l'invite USSD sur mon téléphone.**

Vérifiez que le numéro MoMo saisi est correct et que votre ligne est active. Si l'invite ne vient pas dans les 2 minutes, contactez votre agence avec votre référence.

---

**Q : Je ne me souviens plus de ma référence de souscription.**

Rendez-vous dans une agence Afriland avec votre CNI. Un agent peut retrouver votre dossier par nom, numéro de téléphone ou numéro CNI.

---

**Q : Ma photo (selfie) a été rejetée au point d'impression. Que se passe-t-il ?**

Le point d'impression peut remplacer la photo KYC et planifier un nouveau passage. Contactez votre agence pour prendre rendez-vous.

---

**Q : Je ne peux plus me connecter à mon compte staff.**

Vérifiez vos identifiants. Si le problème persiste, votre compte a peut-être été désactivé ou votre mot de passe a expiré. Contactez votre administrateur système.

---

**Q : Combien de temps après la validation du paiement ma recharge est-elle créditée ?**

La recharge est créditée après que le caissier a confirmé l'opération dans le portail. Ce délai est généralement de quelques heures à une journée ouvrable selon les agences.

---

**Q : Puis-je voir le numéro complet de ma carte dans le portail ?**

Non. Par mesure de sécurité (conformité PCI DSS), le portail n'affiche jamais le numéro complet de la carte. Seuls les 4 premiers et 4 derniers chiffres sont visibles.

---

**Q : L'OCR n'a pas correctement lu mon reçu SARA. Que faire ?**

L'agent ou le caissier peut corriger manuellement les données extraites avant de valider le reçu. Si le reçu est illisible, fournissez un document de meilleure qualité.

---

**Q : Comment récupérer mes données personnelles ?**

Conformément à la réglementation en vigueur, vous pouvez adresser une demande de droit d'accès à vos données à Afriland First Bank — Direction Digital.
